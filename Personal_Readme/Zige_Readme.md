### 无人机浮游植物识别项目 - 交互展示（乔梓阁）专属 README

#### 一、核心定位
作为项目展示端负责人，将所有算法与数模产出集成至本地可运行的交互式 Demo，模拟真实水华监测预警产品的使用流程。

#### 二、绝对清晰的工作边界
✅ **核心负责（前端展示 + 用户交互 + 数据集成）**
1. 展示端框架搭建：基于 Streamlit/Flask 等任选技术，实现本地可运行程序。
2. 池塘管理功能：
   - 创建池塘：录入名称、基准 GPS、飞行高度。
   - 上传数据：按日期上传原始航拍图与无人机参数 JSON。
   - 自动校验：校验上传图像的 GPS 和高度是否在允许偏差内。
3. 预测设置 UI：
   - 简易模式：默认使用统计模型，一键预测。
   - 精准模式：展示参数表格（水温、光照、生长速率等），预填科学默认值，支持手动修改或上传环境数据文件。
4. 核心可视化：
   - 时间进度条切换日期。
   - 实测/预测模式一键切换。
   - 固定底图 + 动态掩码叠加渲染。
   - 侧边栏展示面积、覆盖率、浓度等级等量化数据。
5. 数据来源：从 `outputs/masks/` 读取掩码图，从 `outputs/quantifications/` 读取量化 JSON。

❌ **绝不涉及**
- 不参与数据采集、标注、模型训练、数模计算。

#### 三、交互流程设计

1. **首页**：显示“我的池塘”列表，可创建新池塘。
2. **池塘详情**：
   - 时间轴：水平进度条，标记有实测数据的日期（实心点）和预测日期（空心点）。
   - 主视图：底图 + 当前选中日期的掩码叠加层。
   - 模式切换：顶部开关切换“实测数据” / “预测数据”。
   - 量化面板：右侧或底部显示面积、覆盖率、浓度等级。
3. **预测设置面板**：
   - 入口：池塘详情页的“预测设置”按钮。
   - 简易模式：直接点击“开始预测”。
   - 精准模式：显示参数表格，预填默认值（灰色标注来源），支持修改和文件导入。

#### 四、与上游数据接口约定
- 掩码图路径：`outputs/masks/YYYYMMDD_*_mask.png`（实测）和 `outputs/masks/YYYYMMDD_predicted_mask.png`（预测）。
- 量化数据：`outputs/quantifications/YYYYMMDD_*.json`，包含字段：date, total_area_m2, coverage_ratio, concentration_level, class_areas 等。
- 默认参数：刘俊辉交付的 `params_default.json`，用于预填 UI 参数表格。
