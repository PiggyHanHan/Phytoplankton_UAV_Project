### 无人机浮游植物识别项目 - 数模时序预测（刘俊辉）专属 README

#### 一、核心定位
作为项目数模预测负责人，基于数学建模方法搭建双模型预测系统，输出未来水华趋势与风险等级，并利用 AI 模块提供的空间仿真工具生成预测分布图。

#### 二、绝对清晰的工作边界
✅ **核心负责（双模型预测 + 空间仿真调用 + 默认参数设计）**
1. 从 `outputs/quantifications/` 读取 AI 模块交付的时序实测面积数据。
2. 搭建并实现**两套**水华面积预测模型：
   - **统计基线模型（ARIMA）**：仅依赖历史面积时序，适合无环境数据场景。
   - **机理增强模型（Logistic 生长方程 + 环境驱动）**：基于藻类种群动力学方程，可融合水温、光照等变量。方程参数通过非线性最小二乘法拟合。
3. 编写《模型参数说明书》，为机理模型中的所有参数提供基于文献和地区气候统计数据的默认值，并标注明确来源。
4. 完成两模型的预测误差对比（RMSE/MAE），产出模型对比评估表。
5. 调用吴天宇交付的 `utils/simulation/extract_growth_pattern.py`，从实测掩码序列中提取空间生长规则。
6. 将预测的未来面积值及生长规则传入 `utils/simulation/generate_prediction_mask.py`，生成未来预测掩码图，存入 `outputs/masks/`。
7. 输出标准化预测数值数据至 `outputs/quantifications/`，确保格式与实测数据完全对齐。

❌ **绝不涉及**
- 不参与无人机图像预处理、标注。
- 不参与 AI 模型训练、推理。
- 不开发前端展示界面。

#### 三、模型设计说明

**1. 统计基线模型（ARIMA）**
- 输入：历史日期及对应的浮游植物总面积（来自 `outputs/quantifications/`）。
- 方法：自回归积分滑动平均模型，通过 AIC 准则自动选择最优阶数。
- 输出：未来 1-2 个月的逐日面积预测值。
- 适用：用户未上传任何环境数据时的默认预测方案。

**2. 机理增强模型（Logistic 生长方程）**
- 核心方程：
  dN/dt = r · N · (1 - N/K)
  其中 r = r_max · f(T) · g(L)，T 为水温，L 为光照。
- 参数拟合：使用实测面积序列，采用非线性最小二乘法（如 Levenberg-Marquardt）估计 r_max、K 等参数。若用户提供了实测环境数据，则先拟合 f(T)、g(L) 中的系数；否则使用默认的温度/光照响应曲线。
- 求解：使用数值积分（如 scipy.integrate.odeint）求解微分方程，外推未来面积。
- 适用：用户提供环境数据或同意使用默认环境参数时的精准预测。

**3. 默认参数卡（节选自《模型参数说明书》）**
| 参数 | 默认值 | 单位 | 来源 |
|:---|:---|:---|:---|
| 最大比生长速率 r_max | 0.8 | d⁻¹ | 微囊藻在25°C下的典型值（Reynolds, 2006） |
| 环境容纳量 K | 首次实测面积 × 3 | m² | 保守估计，假设池塘最大承载为当前3倍 |
| 水温 T | 25（夏季）/ 8（冬季） | °C | 华中地区池塘表层水温典型值 |
| 光照 L | 12（夏季）/ 10（冬季） | h | 华中地区平均日照时数 |
| 温度半饱和常数 | 2.0 | °C | 文献默认值 |
| ... | ... | ... | ... |

#### 四、与 AI 模块的协作接口
1. 读取 `outputs/quantifications/` 中的时序 JSON 文件（格式由吴天宇约定）。
2. 调用 `utils/simulation/extract_growth_pattern.py`：
   - 输入：两个连续实测日期的掩码图路径。
   - 输出：生长规则 JSON（包含扩张速率、方向分布、边缘扩张占比等）。
3. 调用 `utils/simulation/generate_prediction_mask.py`：
   - 输入：基准掩码路径、目标面积变化率、生长规则 JSON、输出路径。
   - 输出：预测掩码图存入 `outputs/masks/`。

#### 五、最终交付物
1. ARIMA 模型代码文件。
2. Logistic 生长模型代码文件（含参数拟合与方程求解）。
3. 模型对比评估表（RMSE/MAE）。
4. 《模型参数说明书》（含完整默认参数表及文献来源）。
5. 未来预测面积量化数据（`outputs/quantifications/`）。
6. 仿真预测掩码图（`outputs/masks/`）。
7. 与水华风险等级评估报告。
